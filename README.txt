기초 컴퓨터 그래픽스 - 프로그래밍 숙제 1
========================================

< OpenGL API 함수를 사용한 2차원 기하 변환 연습>

1. 구현

bird 물체를 움직여 다가오는 물체들을 피하는 게임이다.


2. 구조

- 제공 된 물체 중 사용한 물체 및 적용한 변환:

1) airplane
화면 윗 부분에서 위 아래로 진동 움직임을 하며 화면의 오른쪽에서 왼쪽으로 이동한다.

2) house
여러 house 물체가 화면의 오른쪽에서 왼쪽으로 이동한다. 각 house는 크기와 X 좌표가 다르다 (랜덤).

3) cocktail
X 좌표는 고장하고 랜덤으로 정해지는 Y 좌표에서 화면의 오른쪽에서 왼쪽으로 360도 연속으로 회전하며 이동한다.

4) car
airplane 물체에서 중력으로 인해 떨어지는 효과를 보였다. 떨어지기 시작하는 X 좌표는 랜덤이다. ground에 부딛히면 더 이상 아래로 이동하지 않는다.


- 집적 생성한 기하 물체 및 적용한 변환:

1) bird
X좌표를 고정하고 SPACE key 를 누르면 점프하는 효과가 있다. 아무 키를 누르지 않으면 아래쪽으로 이동한다. 또한 이동하는 방향으로 회전을한다.

2) ground
화면 아래 부분에 고정한체 그려진 물체이다.
